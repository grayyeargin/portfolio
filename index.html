Brewfinder

Website:   Brewfinder
Role:   Web Developer

In producing Brewfinder I created a RESTful API using Ruby on Rails and PostgreSQL to serve the client-side with customized JSON responses. The client-side is a one page application using Backbone.js to create an MVC structure, rendering templates for each route. D3.js is used to produce a map and pie chart for query purposes.

MusicMap

Website:   MusicMap
Role:   Web Developer

MusicMap is a web application designed to aggregate information on musical artist's to give the user more information in one search. To do this my team and I consumed multiple 3rd party API's and directed them to one page for each queried artist. This query would produce a page with relevant Twitter, Spotify, lastFM, and Reddit information on the artist. Bootstrap is utilized for page styling with jQuery effects on the album transitions on the home page.

Skills Utilized

  • Ruby on Rails
  • Javascript
  • Bootstrap
  • CSS
  • jQuery
  • API

About Me

Hi, my name is Gray and I am a Web Developer living in Brooklyn, NY. I built my first webpage on Angelfire when I was 12 and have since learned how to code for both the client and server side of web applications.

My experience includes creating complex web applications utilizing Javascript frameworks/libraries/tools such as Backbone.js, jQuery, D3.js, and Underscore.js. I keep my CSS styling dry and organized with Sass and animate elements with CSS3. I have created several server side applications with Ruby, making use of frameworks such as Ruby on Rails and Sinatra. These applications have included creating RESTful API's with a PostgreSQL database and consuming 3rd party API's.

In my spare time away from work I like to enjoy what the great outdoors has to offer, whether its backpacking, kayaking, hitting some tennis balls, or just having a good read at the beach. I also have a passion for music, photography, and horticulture.
